Quiz App is part of the Developer Akademie's training programme for software developers ([www.developerakademie.com](https://www.developerakademie.com)).

## Prerequisites

- [Node.js](https://nodejs.org/) (for installing dependencies)

## Quickstart

Clone the repository:

```bash
git clone https://github.com/karinaklages/quiz-app.git
cd quiz-app
```

Install dependencies:

```bash
npm install
```

Then open `index.html` directly in your browser:

```text
quiz-app/index.html
```

## Project Structure

```text
quiz-app/
├── assets/
│   ├── audio/       # Sound effects
│   ├── icons/       # App icons
│   └── img/         # Images and screenshots
├── scripts/
│   └── data.js      # Quiz question data
├── styles/
│   └── mobile.css   # Responsive styles for mobile devices
├── .gitignore
├── index.html       # Application entry point
├── package.json     # Project metadata and dependencies
├── script.js        # Core app logic
└── style.css        # Main stylesheet
```
